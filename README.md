# AI Academic Advisor

AI Academic Advisor System for University of Jordan

## About

This is a comprehensive web-based academic planning system developed for the University of Jordan's Computer Information Systems (CIS) program. The platform helps students plan their academic journey and enables advisors to efficiently review and approve study plans.

## Features

### For Students
- **Dashboard**: Track academic progress, GPA, credits, and graduation timeline
- **Course Catalog**: Browse and search all 132 credit hours with detailed course information
- **AI Chatbot**: Get personalized course recommendations and academic guidance
- **Notifications**: Stay updated on plan approvals, deadlines, and recommendations

### For Academic Advisors
- **Advisor Dashboard**: Review and approve/reject student study plans
- **Plan Management**: Filter plans by status (Pending, Approved, Rejected)
- **Student Information**: View detailed student requests with course selections

## Technologies

- **React 18** with TypeScript
- **Vite** for build tooling
- **Tailwind CSS** for styling
- **shadcn/ui** for UI components
- **React Router** for navigation
- **React Query** for data management

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/abdulrahman-alsnour/AI-Academic-Advisor---ispm-.git
cd AI-Academic-Advisor---ispm-
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:8080`
